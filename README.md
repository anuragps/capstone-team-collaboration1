# Capstone Project Collaboration Repository

This repository was created as part of the GitHub collaboration activity.

The purpose of this repository is to demonstrate the basic GitHub workflow including:
- Creating a repository
- Adding and committing files
- Simulating collaborator contributions
- Updating repository files through multiple commits

Project Idea:
Analyzing time constraints and learning participation among employed adults using American Time Use Survey data.

Note:
Since this activity was completed individually, collaborator contributions were simulated through multiple commits to demonstrate the GitHub collaborative workflow.
